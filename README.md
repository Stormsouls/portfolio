## Job Application Bot

A system that generates tailored CVs and cover letters in PDF format for each job offer.

**What it does:**
- Reads a job offer and adapts the CV content — profile text, selected experiences, keywords
- Detects the language of the offer (German or English) and generates all documents accordingly
- Two CV formats: visual two-column layout with photo, and ATS-safe single-column for large company portals
- Compresses the photo automatically so output files stay under LinkedIn's 2MB upload limit
- The CV and cover letter for this application were generated by this bot

**Stack:** Python · ReportLab · Pillow · Claude AI

**Files:** `generate_cv.py`, `generate_cv_ats.py`, `generate_cover_letter.py`, `job_data_example.json`

## Import Intelligence Bot

An automated market intelligence system that detects trending products in the US and evaluates their import viability from China to Argentina.

**What it does:**
- Monitors TikTok Shop, Amazon Movers & Shakers (12 categories), Instagram influencer pages, and viral product blogs in real time
- Runs each product through a full import cost model: Alibaba price + air freight + customs VAT + duties + MercadoLibre commission → net margin and score
- Scoring system 0–100 combining demand signals, MercadoLibre competition, margin, and regulatory compliance alerts
- Lists winning products directly on MercadoLibre Argentina via OAuth 2.0 API integration
- Cloudflare bypass using Playwright headless with stealth mode

**Stack:** Python · Playwright · BeautifulSoup4 · MercadoLibre API · Apify

**Files:** `import-bot/evaluador_productos.py`, `import-bot/publicador_meli.py`
