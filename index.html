Import Intelligence Bot
Python · Playwright · MercadoLibre API · Apify
An automated market intelligence system that detects trending products in the US and evaluates their import viability from China to Argentina. Monitors TikTok Shop, Amazon Movers & Shakers, and influencer pages in real time, then runs each opportunity through a full import cost model (Alibaba price + freight + customs + MercadoLibre fees) to calculate net margin and score. Winning products can be listed directly on MercadoLibre Argentina via API.
  • Multi-source trend detection: TikTok Shop (Apify), Amazon Movers & Shakers (12 categories), Instagram influencer link-in-bio pages, and curated viral product blogs
  • Full import cost model: Alibaba estimate + air freight + 21% customs VAT + duties + 15% MeLi commission → net margin and multiplier per product
  • Scoring system 0–100 combining demand signals, competition on MercadoLibre, margin, and regulatory compliance alerts (ANMAT / SENASA / ENACOM)
  • MercadoLibre Argentina OAuth 2.0 integration with auto token refresh and direct listing via API
  • Anti-bot / Cloudflare bypass using Playwright headless with stealth mode
  • Outputs: CSV with all scored opportunities, readable TXT report, and HTML preview styled like MercadoLibre

Cinematic Photo Pipeline
Python · MobileSAM · OpenCV · Folder watcher
An automated photo processing pipeline that turns a raw portrait into a publication-ready image in ~45 seconds — no manual editing required. Drop a photo in a folder, and the system segments the subject using MobileSAM, then applies cinematic hue-selective grading: gold tones survive, everything else desaturates to a grey-slate palette.
  • Folder watcher — drop a photo, get the result automatically
  • MobileSAM semantic segmentation — distinguishes subject from background without color heuristics
  • Hue-selective desaturation: gold survives, everything else becomes cinematic grey-slate
  • Configurable brand palette — swap gold for any hue range
  • Dual mode: fully automatic (watcher) or manual with LLM-generated SAM hints
